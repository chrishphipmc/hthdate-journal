# hthdate-journal

## 项目简介

hthdate-journal 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，而是作为通用页面集合，用于保存和展示各类静态 HTML 内容。

## 目录说明

```
hthdate-journal/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面依赖的资源文件（CSS、图片等）
├── index.html      # 仓库首页或导航页
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件独立存放，文件名应反映内容主题。
- `assets/`：可选的公共资源目录，用于统一管理样式、脚本和媒体文件。

## 页面归档说明

- 所有页面均为静态 HTML 文件，不依赖后端服务。
- 页面内容可包含文本、图片、链接等标准网页元素。
- 归档时请确保文件路径正确，资源引用使用相对路径。
- 建议为每个页面添加简短的注释或说明，便于后续维护。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，并更新 `index.html` 中的导航链接（如有）。
- 更新页面：直接修改对应文件，注意保持路径和资源链接有效。
- 删除页面：移除文件并清理相关引用。
- 欢迎提交 Pull Request 或 Issue 讨论页面归档相关事宜。
